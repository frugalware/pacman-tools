= synclib32(1)

== NAME

synclib32 - updates existing packages for emulation of i686 on x86_64

== SYNOPSIS

synclib32 [options]

== OPTIONS

--help::
	Display this manpage.

--version::
	Display the package version.

== DESCRIPTION

You should use this script, like:

----
$ cd /path/to/source/lib32-extra
$ synclib32
----
